# 商資系課程分析

> 寫來看一下那些課的修課率爆炸

## 安裝

1. 下載至電腦
```
git clone https://github.com/mlgzackfly/nkust-bc-courses.git
```

2. 進入資料夾
```
cd nkust-bc-courses
```
3. 安裝所需套件

```
python3 pip install requirements.txt
```
